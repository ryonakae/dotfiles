## コーディング規約

- レスポンス、コミットメッセージ、コード内コメントは日本語。システムエラーやログの引用は原文のまま
- 既存コードの設計・記法に従う
- コメントは Why だけ。処理内容の説明は書かない

## ツール・環境

- `Operation not permitted` は agent-safehouse の制限。回避策を試みずユーザーに報告する
- Python: CLI ツールは `uvx`、スクリプトは `uv run`、パッケージ追加は `uv add`
- ライブラリ/API のドキュメント参照は `find-docs` スキルを使う
- Web 検索時は一次情報（公式ドキュメント、API リファレンス）を優先する
- Web ページ取得が 403 で失敗したら `agent-browser` を使う
