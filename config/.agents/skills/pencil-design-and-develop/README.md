# Pencil Design and Develop Skill

This skill helps an AI coding agent use Pencil to design, implement, and synchronize UI work with fewer mistakes and less rework.

## What this skill improves

1. Designing and editing `.pen` files with reusable component-first workflows.
2. Translating Pencil designs into implementation-ready code with explicit mapping.
3. Synchronizing code-side updates back into Pencil design structure and styles.
4. Applying safety checks, visual verification, and troubleshooting guidance when issues appear.

## Folder structure

- `SKILL.md`: Entry workflow and mode routing.
- `references/`: Detailed runbooks, safety rules, and quality gates.
