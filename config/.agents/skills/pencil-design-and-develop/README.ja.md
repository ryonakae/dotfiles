# Pencil Design and Develop スキル

このスキルは、AIコーディングエージェントが Pencil を使った設計・実装・同期作業を一貫して進めやすくし、手戻りやミスを減らすためのものです。

## このスキルで改善できること

1. 再利用コンポーネントを前提にした `.pen` ファイルの設計・編集。
2. Pencil デザインから実装コードへの、対応関係が明確な変換。
3. コード側の構造やスタイル変更の Pencil デザインへの同期。
4. 問題発生時の安全チェック・視覚検証・トラブルシューティング。

## ディレクトリ構成

- `SKILL.md`: エントリーフローとモード振り分け。
- `references/`: 詳細ランブック、セーフティルール、品質ゲート。
