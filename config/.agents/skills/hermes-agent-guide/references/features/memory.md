# メモリシステム

> 参照元: https://hermes-agent.nousresearch.com/docs/user-guide/features/memory

2つのファイルで構成:

| ファイル | 用途 | 文字制限 |
|---------|------|--------|
| `MEMORY.md` | エージェントノート（環境・慣例・学習） | 2,200文字 (~800トークン) |
| `USER.md` | ユーザープロフィール（設定・スタイル） | 1,375文字 (~500トークン) |

## memory ツールアクション

- `add` -- 新規エントリ追加
- `replace` -- 既存エントリ更新（`old_text` 部分文字列マッチ）
- `remove` -- エントリ削除（`old_text` 部分文字列マッチ）

セッション開始時にシステムプロンプトに凍結スナップショットとして挿入。変更はディスクに即時保存されるが、システムプロンプトは次セッションまで更新されない。
